Firefox について

世界中で人気爆発中のパソコン向けフルブラウザ「Firefox」

Firefox は、インターネットをもっと自由に、快適に、そして安全に使えたら、という思いから生まれた新しいブラウザソフトです。便利な機能をいち早く使えて、カスタマイズも自由自在。これはもう、乗り換えるしかないかも！？

無料ダウンロード Windows 版 (5.7 MB)
詳しくはこちら - その他のシステムと言語

便利で使いやすい！

たくさんのページを同時に開いておける「タブブラウズ機能」を使えば、お気に入りのサイトもサクサク見て回れます。Google や Yahoo! JAPAN でいつでも検索できるよう、ウィンドウの右上には検索窓も付けました。ニュースやブログの新着記事も簡単に読めるようになっています。

セキュリティもバッチリ！

Firefox はセキュリティの高さで定評があります。ウイルス、スパイウェア、フィッシング詐欺の被害を未然に防ぎ、セキュリティホールが見つかった場合も専門家と協力してすばやく対応します。プライバシーを守るために、履歴を瞬時に消去できる機能も搭載しています。

自分好みにカスタマイズできる！

検索ツールバー、メールの着信通知、動画ダウンローダー、多言語辞書など、欲しい機能を好きなだけ追加できる、無料の「拡張機能」が充実。気分に合わせて Firefox のデザインを変えられる「テーマ」も選べます。自分だけのオリジナルブラウザを作って、友達に差を付けよう！

Firefox に興味が湧いたら、GetFirefox.jp へ - Firefox の最新版に関する詳細やサポート情報、拡張機能など

おすすめ拡張機能

Google ツールバー
Internet Explorer でもおなじみの人気ツールバー。Gmail や Google ドキュメントとの連携も。
タブカタログ
複数のタブに開いているページの内容をサムネイルで一覧表示できるようにします。
ScrapBook
Web ページの一部または全体を保存して、コレクションとして整理、編集できます。
Firebug
ワンクリックで使える豊富な開発ツールを統合した、Web デザイナーのための「最終兵器」です。

Firefox って？

皆さんはインターネットを見るときに青い「e」のアイコンをクリックしていませんか？ Firefox は、その「e」の代わりとなる、便利で安全性の高いフリーソフトです。利用者は世界で 1 億人以上。隣の人はもう使ってるかも？

あなたも参加できます

Firefox は、一般の企業が販売しているソフトと違って、非営利団体を中心にオープンな環境で作られています。開発からプロモーションまで、大勢の皆さんがボランティアとして関わっています。あなたも、家族や友達に Firefox を勧めてくれるだけで、立派なコミュニティのメンバーです。

Firefox と
Creative Commons

Firefox は、オープンソースという方法でソフトの設計図を公開し、自由に利用できるようにしています。Creative Commons (CC) は、それと同じように、クリエイターが自分の作品について、より柔軟に利用条件を設定できる著作権ライセンスです。CC を使って世界に作品を発信してみませんか？
